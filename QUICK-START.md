# ⚡ Quick Start - Telegram Auto Gift Buyer

**Get started in 5 minutes!**

---

## 🚀 Super Fast Setup

### 1. Download (30 seconds)

[![Download](./assets/download-button.svg)](https://github.com/yourusername/telegram-auto-gift-buyer/releases/latest)

Click the button above → Download ZIP file → Save to your computer

---

### 2. Extract (15 seconds)

- Right-click the ZIP file
- Choose "Extract All..."
- Pick a folder
- Click "Extract"

---

### 3. Run (10 seconds)

- Open the extracted folder
- Double-click **TelegramAutoGiftBuyer.exe**
- Click "More info" → "Run anyway" (if Windows SmartScreen appears)

---

### 4. Connect Account (2 minutes)

1. Click **"Add Account"**
2. Enter your phone number: `+1234567890`
3. Click **"Send Code"**
4. Check Telegram app for code
5. Enter the code
6. Done! ✅

---

### 5. Send Your First Gift (1 minute)

1. Click **"Send Gift"** button
2. Select recipient from list
3. Choose gift type
4. Click **"Confirm & Send"**
5. Gift sent! 🎁

---

## 🎯 Common First Tasks

### Schedule a Birthday Gift

```
1. Click "Scheduler" tab
2. Click "New Schedule"
3. Pick date and time
4. Select recipient and gift
5. Save
```

### Send Gifts to Multiple People

```
1. Go to "Bulk Operations"
2. Select recipients (or import list)
3. Choose gift
4. Click "Send to All"
```

### Track Spending

```
1. Click "Analytics" tab
2. View charts and reports
3. Export if needed
```
