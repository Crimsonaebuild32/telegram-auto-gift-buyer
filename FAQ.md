## 🔧 Installation & Setup

### How do I install the software?

1. Download the ZIP file from [Releases](https://github.com/yourusername/telegram-auto-gift-buyer/releases)
2. Extract the archive to your preferred location
3. Run `TelegramAutoGiftBuyer.exe`
4. Follow the setup wizard

**Detailed guide:** [Installation Guide](./INSTALLATION.md)

### What are the system requirements?

**Minimum:**
- Windows 10 (64-bit) Build 19041+
- 4 GB RAM
- 100 MB free space
- Internet connection

**Recommended:**
- Windows 11
- 8 GB RAM
- SSD storage
- High-speed internet

### Why does Windows SmartScreen block the app?

This is a common security feature for new software. To proceed:
1. Click "More info"
2. Click "Run anyway"

We're working on code signing certificates to eliminate this warning.

### My antivirus flags the application. Is it safe?

Yes, it's safe! Some antivirus programs flag new software. The application is:
- ✅ Virus-free (scan with VirusTotal)
- ✅ No malware or spyware
- ✅ Open source code available
- ✅ Digitally signed (upcoming)

**Whitelist the application** in your antivirus settings.

### Can I install it on multiple computers?

Yes! Install on as many devices as you need. Your settings can be synced via cloud backup feature.

### Where is my data stored?

Data is stored locally on your computer at:
```
C:\Users\YourName\AppData\Roaming\TelegramAutoGiftBuyer\
```

Includes:
- Configuration files
- Account sessions
- Transaction history
- Logs

### How do I uninstall?

1. Close the application
2. Delete the installation folder
3. (Optional) Delete `%APPDATA%\TelegramAutoGiftBuyer` folder
4. (Optional) Remove desktop shortcut

No registry changes or system modifications to clean up.

### How many Telegram accounts can I add?

Unlimited! Add as many accounts as you need. The software supports multi-account management with easy switching.

No! Works with both:

All simultaneously.

## 🛠️ Technical Issues

### The application won't start. What do I do?

**Troubleshooting steps:**

1. **Run as Administrator**
   - Right-click exe → "Run as administrator"

2. **Install Visual C++ Redistributables**
   - [Download here](https://aka.ms/vs/17/release/vc_redist.x64.exe)

3. **Update Windows**
   - Install all pending updates

4. **Check antivirus**
   - Whitelist the application

5. **Reinstall**
   - Complete uninstall → fresh install

### I can't connect to my Telegram account

**Common solutions:**

1. **Verify phone number format**
   - Include country code: +1234567890
   - No spaces or dashes

2. **Check internet connection**
   - Test on telegram.org

3. **Telegram servers status**
   - Check downdetector.com

4. **Firewall/proxy**
   - Allow app through firewall
   - Configure proxy if needed

5. **Try different account**
   - Test with another number

### Gifts are not sending

**Diagnostic steps:**

1. **Check account balance**
   - Verify payment method

2. **Verify recipient**
   - Ensure not blocked
   - Account still active

3. **Check logs**
   - Settings → Logs
   - Look for error messages

4. **Test connectivity**
   - Tools → Connection Test

5. **Contact support**
   - If issue persists

### The application is slow or freezing

**Performance fixes:**

1. **Close unnecessary programs**
   - Free up RAM

2. **Reduce concurrent operations**
   - Lower bulk send limit

3. **Clear cache**
   - Settings → Advanced → Clear Cache

4. **Update to latest version**
   - Help → Check for Updates

5. **Check system resources**
   - Task Manager → Performance

### I forgot my master password

**Recovery options:**

1. **Use recovery code** (if set during setup)
   - Settings → Security → Recover Password

2. **Reset application** (loses all data)
   - Uninstall completely
   - Delete `%APPDATA%\TelegramAutoGiftBuyer`
   - Reinstall

**Important:** Master password cannot be recovered. Always save your recovery code!

### How do I view error logs?

**Access logs:**
1. Settings → Logs
2. Click "Open Log Folder"
3. View `application.log` file

**Or:**
```
%APPDATA%\TelegramAutoGiftBuyer\logs\
```

Send logs to support when reporting issues.

### The interface looks weird/broken

**UI fixes:**

1. **Reset layout**
   - Settings → Interface → Reset Layout

2. **Change theme**
   - Try different theme (Light/Dark)

3. **Adjust DPI scaling**
   - Windows Settings → Display → Scale

4. **Update graphics drivers**
   - Latest GPU drivers

5. **Reinstall application**
